# Customer Churn Prediction

## Overview
This project builds a machine learning model to predict customer churn using demographic, service, and billing data.

## Objectives
- Perform exploratory data analysis (EDA)
- Train classification models to predict churn
- Evaluate models using accuracy, precision, recall, and F1-score
- Practice production-style ML project structure and Git workflows

## Dataset
The project uses a public customer churn dataset.

## Tech Stack
- Python
- pandas
- scikit-learn
- matplotlib / seaborn
